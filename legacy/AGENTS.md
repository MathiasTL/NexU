# AGENTS.md — Reglas para agentes de IA en este repo

## Git

- **NO hagas commits.** Los miembros del equipo hace todos los commits manualmente. Tu trabajo es dejar los cambios en el working tree (staged o unstaged) y avisarle qué archivos modificaste.
- No corras `git commit`, `git push`, `git merge`, `git rebase` ni ningún comando que escriba al historial.
- Sí podés correr comandos read-only (`git status`, `git diff`, `git log`).
